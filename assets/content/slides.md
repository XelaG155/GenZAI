# Slide Deck Outlines – GenZAI

Jede Deck-Struktur nutzt 16:9 Slides. Kopiere die Stichpunkte in Google Slides/Canva/Keynote. Tone: klar, wenig Text, viel Visuals.

## Deck 1: "AI 101 für Macher"
1. Hook: "Warum jetzt?" (Hype vs. Value)
2. Modelle heute: GPT-4o, Claude 3.5, Llama 3 (Open vs. Closed)
3. Was kann ich bauen? (Chat, Vision, Agents, RAG)
4. Prompt Patterns (CoT, Few-Shot, Guardrails)
5. Risiken: Bias, Halluzinationen, PII
6. Mini-Demo: Prompt → Response mit Annotation
7. Call-to-Action: Starte Lab 1

## Deck 2: "RAG in 5 Schritten"
1. Problem: Kontext fehlt → Halluzination
2. Pipeline: Ingest → Embed → Store → Retrieve → Generate
3. Datenvorbereitung: Chunking, Dedup, Metadata
4. Retrieval Tuning: k, Score-Threshold, Rerank
5. Antwortformat: Zitate, Quellenpflicht
6. Live Metriken: Precision/Recall, Latenz, Kosten
7. Demo-Flow & Checkliste
8. Nächste Steps: Eval Suite + Caching

## Deck 3: "Agents & Tools"
1. Wann Tools? (Daten holen, Aktionen ausführen)
2. Tool-Design: JSON-Schema, Fehlerfälle
3. Loop Control: Max Steps, Fallback, Timeouts
4. Beobachtbarkeit: Logs, Traces, Replay
5. Sicherheit: Allowlist, PII-Filter, Abuse-Kontrolle
6. Demo: Wetter + Kalender Agent
7. QA: Regression Prompts

## Deck 4: "Infra & Deploy"
1. Laufzeit-Optionen: Cloud API vs. Self-Host (VM/GPU/Serverless)
2. Setup: Ubuntu, Docker, Compose
3. Netzwerk: Ports, Reverse Proxy, TLS
4. Monitoring: Logs, Metrics, Alerts
5. Kosten: Token + Infra + Support
6. Playbook: Dev → Staging → Prod
7. Disaster Recovery: Backups, Rollbacks

## Deck 5: "Growth & Biz"
1. Value Prop in 1 Minute
2. Pricing Ladder (Free/Pro/Team)
3. Onboarding-Flow: 60s zur ersten guten Antwort
4. Marketing: Reels, Live-Build, Community
5. Metriken: Activation, Retention, CAC/LTV
6. Support & Trust: Status Page, Changelog, Policy
7. Launch Timeline (Week 1-4)

## Bonus Deck: "AI Safety Lite"
1. Safety Risiken (PII, Jailbreaks, NSFW)
2. Filter + Guardrails
3. Red-Teaming Beispiele
4. Consent & Logging
5. Governance: Rollen, Zugriffe, Reviews
