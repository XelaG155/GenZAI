# GenZAI

Ein kompletter AI-Kurs für Gen Z. Öffne `index.html` für Curriculum, Labs, detaillierte Übungen + Lösungen, Infrastruktur-Guides, Business-Playbooks sowie verlinkte Slides & Projekt-Briefs.

## Quickstart
1. Repo klonen oder herunterladen.
2. `index.html` im Browser öffnen.
3. Teile oder passe die Inhalte (Use Cases, Übungen, Launch-Plan, Slides, Projekt-Briefs) für deine Community an.
